# نظام إدارة مكاتب الاستقبال مع حجز مواعيد وخدمات دفع
==========================

## ملخص المشروع

نظام إدارة مكاتب الاستقبال هو تطبيق إلكتروني يهدف إلى تسهيل إدارة المكاتب الاستقبال، بما في ذلك حجز مواعيد وخدمات دفع. يهدف المشروع إلى توفير تجربة مستخدمة سهلة ومتكاملة لعمليات إدارة المكاتب الاستقبال.

## هيكل المشروع

### مجلدات المشروع

* `app`: يحتوي على كود التطبيق الرئيسي.
* `docker`: يحتوي على ملفات docker-compose و Dockerfile.
* `docs`: يحتوي على dokumentations و README.
* `tests`: يحتوي على اختبارات الوحدة والوظائف.

### جدول المكونات

| مجلد | وظيفة |
| --- | --- |
| `app` | كود التطبيق الرئيسي |
| `docker` | ملفات docker-compose و Dockerfile |
| `docs` | dokumentations و README |
| `tests` | اختبارات الوحدة والوظائف |

## إعداد بيئة التطوير

### استخدم docker-compose

1. تأكد من أنك تمتلك docker-compose مثبتًا على جهازك.
2. انتقل إلى مجلد المشروع واكتب الأمر التالي في командية terminal:
bash
docker-compose up

3. سيتم تشغيل containers وسيتم تشغيل التطبيق على `http://localhost:8000`.

### إعداد البيئة

* تأكد من أنك تمتلك قاعدة بيانات PostgreSQL مثبتة على جهازك.
* تأكد من أنك تمتلك قاعدة بيانات MySQL مثبتة على جهازك.

## مكونات النظام

### أجزاء النظام

* `users`: يحتوي على معلومات المستخدمين.
* `appointments`: يحتوي على معلومات الحجوزات.
* `payments`: يحتوي على معلومات الخدمات الدفعية.

### أدوار النظام

* `admin`: يمتلك صلاحيات إدارة النظام.
* `user`: يمتلك صلاحيات استخدام النظام.
